**PingForest** is a lightweight and efficient network monitoring solution designed to manage IT assets in a hierarchical tree structure. It provides real-time visibility into your network infrastructure by periodically checking device availability and service status.

<br>

### Key Features
* **Hierarchical Management:** Add, remove, and organize devices in a logical tree structure (e.g., Data Center > Rack > Server).
* **Connectivity Monitoring:** Periodically checks device status using **ICMP Ping**.
* **Service Verification:** Monitors specific **TCP Port** availability (HTTP, SSH, DB, etc.).
* **Status Dashboard:** Provides a clear overview of the network health at each node level.
